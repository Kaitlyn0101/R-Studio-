About the assignment.

In this assignment you will learn to bring in a dataset using the url. The assignment is in a RMarkdown file and give you a glimpse of working with RMarkdown. We will learn how to work with RMarkdown in details in our subsequent class. The advantage of a Rmarkdown file is that you can put your narrative along with the code and output in the same file. This cannot be done in an .R script without using the #comment. For this assignment, in the sections where you will need the R code to produce the required results, the blank R code chunks have been inserted. Write the R code in those sections as you would have done in the R console. Use the green triangle in the top right corner of each chunk to run your code or you can run the code in the console window and insert it in the code chunk thereafter. Once all the codes have been inserted, run the file and knit it thereafter. Please remember to change the author name to your group number at the beginning of the file. The html file is for your reference. The knitted html file should be similar to the one provided in D2L but will have the appropriate group number in the author section.

Essential details

Deadline and submission

The M1 ICA2 is an in-class group assignment. However, each member of the group who worked together in class will submit their in-class assignment at the end of the class time. Submit your work by uploading both your RMD and HTML/PDF files through D2L. Although the expectation is to get it done within the class period, if a group needs a little bit of extra time to finish, they can submit the assignment on or before 11 PM of Sept 8,2021. Late work after this deadline will not be accepted except under certain extraordinary circumstances.

Grading

ICAs are graded for participation and completion. Each team member who was present in class and worked on the ICA should submit their files. To grade the ICAs, the instructional team will pick one submission at random from each group. Thus it is important that there is good intra-group communication and teamwork. Each group should ensure that everyone in the team understood, worked through and completed the assignment.
